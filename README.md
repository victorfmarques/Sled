# Sled
Repo designed to Django + Django Rest Framework studies.

## Description
E-commerce application, where users can buy or sell products

## Features
  - Users
    - Profile Picture
    - Address register
    - Purchase history
    - Sales history 
    - Sales control
    - Inventory control
    - Rating
    
  - Products
    - Names and Attributes
    - Category
    - Pictures
  
  - Sales
    - Descriptions
    - Price
    - Quantity 
    - Delivery fee
    - Type of payment
